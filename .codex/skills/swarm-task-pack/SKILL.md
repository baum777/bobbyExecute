# Swarm Task Pack

Break multi-agent work into clean, reviewable task packets with minimal overlap and explicit ownership.

## Objectives

- Assign clear roles and boundaries.
- Preserve dependency order and handoff clarity.
- Keep outputs mergeable and reviewable.
- Prevent multiple agents from freelancing on the same scope.

## Workflow

1. Define the shared goal and the constraints all agents must respect.
2. Split the work into role-bounded tasks with explicit ownership.
3. Identify dependencies, sequencing, and required handoffs.
4. Define acceptance criteria for each role.
5. Describe how results should be merged, reviewed, and reconciled.

## Output Format

- Overall goal
- Shared constraints
- Agent roles
- Task per role
- Dependency order
- Acceptance criteria
- Merge and review flow
- Final reconciliation notes

## Guardrails

- Avoid overlapping execution authority unless it is intentional and justified.
- Separate planning from implementation and review whenever possible.
- Keep each agent task concrete enough to execute without extra interpretation.
